Voice-Controlled Car Dashboard

A web-based car control simulator where users can control AC, lights, windows, temperature, and music using voice commands or typed text.
The backend is built with Java Spring Boot, and the frontend uses React with the Web Speech API.

Technologies Used
Frontend
1)React 18
2)Web Speech API (SpeechRecognition)
3)SpeechSynthesis API
4)JavaScript (ES6+)
5)CSS

Backend

1)Java 17+
2)Spring Boot 3+
3)REST API
4)DTOs + Service Layer Architecture
